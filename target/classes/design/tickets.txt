
PCMS-1 As a User of the system, I should be able to add a new veterinarian.
I should be able to specify: first name, last name, address, speciality.

PCMS-2 As a User should be able to see all the vets.I should be able to see for each vet
first name, last name, address, speciality.

PCMS-3 As a User, I should be able to delete and existing vet by id.

PCMS-4 As a User, I should be able to update an existing vet by id. I should be able to update first name, last name, address, speciality.

PCMS-5 As a User, I should be able to add a new pet. I should be able to specify: race, birthdate, is vaccinated
and owner name.

PCMS-6 As a User should be able to see all the pets.I should be able to see for each pet
      race, birthdate, is vaccinated and owner name.

PCMS-7 As a User, I should be able to delete an existing pet by id.

PCMS-8 As a User I should be able to update an existing pet by id.
 I should be able to update:race, birthdate, is vaccinated, and owner name.

PCMS-9 As a User, I should be able to add a new consult. I should be able
to specify vet, pet, date, description

PCMS-10 As a User, I should be able to view all the consults. I should be able to
see each consult: id, vet_id, pet_id, date, description.

PCMS-11 As a User, I should be able to update the description of a consult.
